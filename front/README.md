## Start the project

Install dependencies:

``` npm install ```

Launch Front-end:

``` npm run start ```


## Tests

### E2E tests

Launching e2e test:

``` npm run e2e:ci ```

Generate coverage report (you should launch e2e test before):

``` npm run e2e:coverage ```

Report is available here:

> front/coverage/lcov-report/index.html

### Unitary tests

Launching test:

``` npm run test ```

for following change:

``` npm run test:watch ```

for coverage:

``` npm test -- --coverage ```
